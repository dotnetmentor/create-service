# create-service

Scaffolding script for new services

Features:

  - Default readme
  - Common dotfiles (.gitignore .editorconfig etc)
  - Default directory structure
  - Koshu setup (taksrunner)
  - Git setup
  - Github setup

## Usage

    ./create-service <name>

## Roadmap

  - Buildkite setup (pipeline.yml, webhook, status badge)
